#  ABC 087 B - Coins
## 問題概要
500円玉をA枚、100円玉をB枚、50円玉をC枚持っています。これらの硬貨の中から何枚かを選び、合計金額をちょうどX円にする方法は何通りあるでしょうか？

## 制約
* 0≤A,B,C≤50
* A+B+C≥1
* 50≤X≤20000
* A,B,Cは整数である
* Xは50の倍数である

## 入力
``` bash
A
B
C
X
```

## 数値例
* A=2
* B=2
* C=2
* X=100
　
答え: 2

条件を満たす選び方は以下の 2 通りです。

* 500 円玉を 0 枚、100 円玉を 1 枚、50 円玉を 0 枚選ぶ
* 500 円玉を 0 枚、100 円玉を 0 枚、50 円玉を 2 枚選ぶ
